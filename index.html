Testing examples

Jasmine, PHPUnit, Behat, Cucumber

Jasmine Overview

  • Jasmine BDD
  • RSpec-style describe...it syntax
  • Large matcher library
  • Straight-up JavaScript
  • Abstracts dependencies with "spies"
  • Supports async operations
  • Extensible with plugins

Jasmine Deets

  • uses expect(value).toBe(value) style assertions
  • Robust matcher library: .toBe(), .toEqual(), .toMatch(expr), .toBeTruthy(), .toThrow()
  • Fluent .not modifier (e.g. expect(4).not.toEqual(5))
  • Match against type using jasmine.Any()
  • jQuery specific matchers provided by jasmine-jquery plugin

Supports async


describe("System object", function() {

	it("does behavior", function() {
		var someVal = false;
		runs(function() {
			setTimeout(1000, function() {someVal = true;});
		});
		
		waitsFor(function() {
			return someVal;
		}, "waiting for value to return true", 1010);
	
		runs(function() {
			expect(someVal).toEqual(true);
		});
	});
	
});

Jasmine (Old way)

  • Separate test runner and spec file
  • Separate VHost for running tests
  • Auto-generated fixtures from Zend_View .phtml
  • Separate file structure under tests/unit/javascript/
  • Creative Spec

Some things to note

  • Single spec can be isolated
  • $.AJAX spies can be used to isolate callbacks Example

Jasmine (New way)

  • Spec files in same file structure as source files
  • No need to generate a test runner
  • Dependencies are declared through docblock comments (@bundle, @dependencies, file name)
  • New in-app test dashboard

PHPUnit

  • Straight port of JUnit to PHP
  • Has evolved over time. This is both good and bad.
  • Some features (mocking, bdd) feel tacked on.

Behat and Mockery

Possible improvements to PHP testing

  • True BDD tool
  • More elegant, streamlined mocking

Jenkins

  • AppNexus Jenkins Dashboard

PhantomJS

  • PhantomJS
  • Runs tests fast on the command line
  • JavaScript API for manipulating the browser (e.g. loading pages, releasing memory, listening to browser events, etc.)
  • Ability to unit test traditionally difficult tasks like file uploads into forms
